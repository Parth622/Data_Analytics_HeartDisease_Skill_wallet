Documentation Workbook
Technical notes and implementation details

Project Objective

Deliver a Tableau-style heart disease analytics portal with interactive BI workflows powered by Flask, SQLite, Pandas, and Plotly.

Dataset Description

File: Heart_new2.csv. The dataset includes health and lifestyle indicators such as smoking, alcohol use, activity, age bands, and heart disease label.

Data Preparation Process

Records are de-duplicated, missing values are imputed, numeric fields are normalized, and derived fields are generated: Age Group, Cholesterol Category, Blood Pressure Category, Lifestyle Risk Score.

SQL Analysis

SQLite stores processed records in heart_data. Query modules return totals, prevalence, average cholesterol estimate, average age, gender distribution, smoking impact, and alcohol impact.

Dashboard Explanation

The dashboard contains KPI cards, a Tableau-like filter pane, and 10 interactive sheets. All visualizations update dynamically based on selected filters.

Story Insights

The story view presents narrative scenes around age risk, lifestyle factors, and gender trends to support business communication and interpretation.

Performance Testing

A dedicated page reports row count, number of visualizations, calculated fields, and average SQL query execution time.

Future Improvements

Add authentication, role-based access, model scoring APIs, and deployment on managed cloud services with CI/CD.
